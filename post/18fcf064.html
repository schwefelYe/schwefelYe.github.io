From State Value to Action Value

  • State value: The average return the agent can get starting from a state.
  • Action value: The average return the agent can get starting from a state and taking an action.

Why Do We Care About Action Value?

Because we want to know which action is better.

Definition of action value

The definition of action value is as follows：

qπ(s,a)=E[Gt∣St=s,At=a]q_{\pi}(s, a) = \mathbb{E}[G_t|S_t = s, A_t = a] qπ​(s,a)=E[Gt​∣St​=s,At​=a]

  • qπ(s,a)q_{\pi}(s, a)qπ​(s,a) is a function of the state-action pair (s,a)(s, a)(s,a)
  • qπ(s,a)q_{\pi}(s, a)qπ​(s,a) depends on π\piπ

Relationship between action value and state value

It follows from the properties of conditional expectation that

E[Gt∣St=s]⏟vπ(s)=∑a∈AtE[Gt∣St=s,At=a]π(a∣s)⏟qπ(s,a)\begin{aligned} \underbrace{\mathbb{E}\left[G_t\mid S_t=s\right]}_{v_\pi(s)} = \sum_{a\in\mathcal{A}_t} \underbrace{\mathbb{E}\left[G_t\mid S_t=s,A_t=a\right]\pi(a\mid s)}_{q_{\pi}(s,a)} \end{aligned} vπ​(s)E[Gt​∣St​=s]​​=a∈At​∑​qπ​(s,a)E[Gt​∣St​=s,At​=a]π(a∣s)​​​

Hence,

vπ(s)=∑aπ(a∣s)qπ(s,a)(1)\textcolor{hotpink}{v_{\pi}(s)} = \sum_a \pi(a|s)\textcolor{hotpink}{q_{\pi}(s, a)} \tag{1} vπ​(s)=a∑​π(a∣s)qπ​(s,a)(1)

Recall the Bellman equation, the state value is given by

vπ(s)=∑a∈Aπ(a∣s)[∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vπ(s′)]⏟qπ(s,a)v_{\pi}(s) = \sum_{a \in A} \pi(a | s) \underbrace{\left[ \sum_{r} p(r | s, a) r + \gamma \sum_{s'} p(s' | s, a) v_{\pi}(s') \right]}_{q_{\pi}(s,a)} vπ​(s)=a∈A∑​π(a∣s)qπ​(s,a)[r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vπ​(s′)]​​

By comparing (1) and this equation, we have the action-value function as

qπ(s,a)=∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vπ(s′)(2)q_{\pi}(s, a) = \sum_{r} p(r | s, a) r + \gamma \sum_{s'} p(s' | s, a) v_{\pi}(s')\tag{2} qπ​(s,a)=r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vπ​(s′)(2)

Remarks：

（1）and（2）are the two sides of the same coin：

  • （1）shows how to obtain state values from action values
  • （2）shows how to obtain action values from state values
