state values

Some notations

Consider the following single-step process：

St→AtRt+1,St+1S_t \xrightarrow{A_t} R_{t+1}, S_{t+1} St​At​​Rt+1​,St+1​

  • t,t+1t, t + 1t,t+1：discrete time instances
  • StS_{t}St​：state at time ttt
  • AtA_{t}At​：the action taken at state StS_{t}St​
  • Rt+1R_{t+1}Rt+1​：the reward obtained after taking AtA_{t}At​

Note that St,At,Rt+1S_t, A_t, R_{t+1}St​,At​,Rt+1​ are all random variables.

This step is governed by the following probability distributions:

  • St→AtS_t \to A_tSt​→At​ is governed by π(At=a∣St=s)\pi(A_t = a \mid S_t = s)π(At​=a∣St​=s)
  • St,At→Rt+1S_t, A_t \to R_{t+1}St​,At​→Rt+1​ is governed by p(Rt+1=r∣St=s,At=a)p(R_{t+1} = r \mid S_t = s, A_t = a)p(Rt+1​=r∣St​=s,At​=a)
  • St,At→St+1S_t, A_t \to S_{t+1}St​,At​→St+1​ is governed by p(St+1=s′∣St=s,At=a)p(S_{t+1} = s' \mid S_t = s, A_t = a)p(St+1​=s′∣St​=s,At​=a)

Consider the following multi-step trajectory：

St→AtRt+1,St+1→At+1Rt+2,St+2→At+2Rt+3,…S_t \xrightarrow{A_t} R_{t+1}, S_{t+1} \xrightarrow{A_{t+1}} R_{t+2}, S_{t+2} \xrightarrow{A_{t+2}} R_{t+3}, \ldots St​At​​Rt+1​,St+1​At+1​​Rt+2​,St+2​At+2​​Rt+3​,…

The discounted return is

Gt=Rt+1+γRt+2+γ2Rt+3+⋯G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots Gt​=Rt+1​+γRt+2​+γ2Rt+3​+⋯

  • γ∈[0,1)\gamma \in [0, 1)γ∈[0,1) is a discount rate.
  • GtG_tGt​ is also a random variable since Rt+1,Rt+2,…R_{t+1}, R_{t+2}, \ldotsRt+1​,Rt+2​,… are random variables.

Definition of state value

The expectation（or called expected value or mean）of GtG_{t}Gt​ is defined as the state-value function or simply state value:

vπ(s)=E[Gt∣St=s]v_{\pi}(s) = \mathbb{E}[G_t \mid S_t = s] vπ​(s)=E[Gt​∣St​=s]

Remarks：

  • It is a function of sss. It is a conditional expectation with the condition that the state starts from sss.
  • It is based on the policy π\piπ. For a different policy, the state value may be different.
  • It represents the “value” of a state. If the state value is greater, the the policy is better because greater cumulative rewards can be obtained.

Q：What is the relationship between return and state value？

A：The state value is the mean of all possible returns that can be obtained starting from a state. If everything π(a∣s),p(r∣s,a),p(s′∣s,a)\pi(a \mid s), \quad p(r \mid s, a), \quad p(s' \mid s, a)π(a∣s),p(r∣s,a),p(s′∣s,a) is deterministic, then state value is the same as return.

Bellman equation

The Bellman equation is a set of linear equations that describe the relationship among the values of all states.

Derivation of Bellman equation

Consider a random trajectory：

St→AtRt+1,St+1→At+1Rt+2,St+2→At+2Rt+3,…S_t \xrightarrow{A_t} R_{t+1}, S_{t+1} \xrightarrow{A_{t+1}} R_{t+2}, S_{t+2} \xrightarrow{A_{t+2}} R_{t+3}, \ldots St​At​​Rt+1​,St+1​At+1​​Rt+2​,St+2​At+2​​Rt+3​,…

While the return GtG_{t}Gt​ can be written as

Gt=Rt+1+γRt+2+γ2Rt+3+…,=Rt+1+γ(Rt+2+γRt+3+…),=Rt+1+γGt+1,\begin{aligned}G_{t}&=R_{t+1}+\gamma R_{t+2}+\gamma^2R_{t+3}+\ldots,\\&=R_{t+1}+\gamma(R_{t+2}+\gamma R_{t+3}+\ldots),\\&=R_{t+1}+\gamma G_{t+1},\end{aligned} Gt​​=Rt+1​+γRt+2​+γ2Rt+3​+…,=Rt+1​+γ(Rt+2​+γRt+3​+…),=Rt+1​+γGt+1​,​

Then, the state value can be written as

vπ(s)=E[Gt∣St=s]=E[Rt+1+γGt+1∣St=s]=E[Rt+1∣St=s]+γE[Gt+1∣St=s].\begin{aligned}v_{\pi}(s)&=\mathbb{E}[G_{t}|S_{t}=s]\\&=\mathbb{E}[R_{t+1}+\gamma G_{t+1}|S_{t}=s]\\&=\textcolor{skyblue}{\mathbb{E}[R_{t+1}|S_{t}=s]}+\gamma\textcolor{hotpink}{\mathbb{E}[G_{t+1}|S_{t}=s]}.\end{aligned} vπ​(s)​=E[Gt​∣St​=s]=E[Rt+1​+γGt+1​∣St​=s]=E[Rt+1​∣St​=s]+γE[Gt+1​∣St​=s].​

The two terms which are highlighted in the last line of are analyzed below.

First term: the mean of immediate rewards

First, calculate the first term E[Rt+1∣St=s]\mathbb{E}[R_{t+1}|S_{t}=s]E[Rt+1​∣St​=s] ：（This term is the mean of immediate rewards.）

E[Rt+1∣St=s]=∑a∈A(s)π(a∣s)E[Rt+1∣St=s,At=a]=∑a∈A(s)π(a∣s)∑rp(r∣s,a)r.\mathbb{E} \left[ R_{t+1} \mid S_{t} = s \right] = \sum_{a \in A(s)} \pi(a \mid s) \mathbb{E} \left[ R_{t+1} \mid S_{t} = s, A_{t} = a \right] = \sum_{a \in A(s)} \pi(a \mid s) \sum_{r} p(r \mid s, a) r. E[Rt+1​∣St​=s]=a∈A(s)∑​π(a∣s)E[Rt+1​∣St​=s,At​=a]=a∈A(s)∑​π(a∣s)r∑​p(r∣s,a)r.

Explanation

  1. From the law of total expectation: given discrete random variables XXX and YYY​, then

E(X)=E(E(X∣Y))E(X) = E(E(X \mid Y)) E(X)=E(E(X∣Y))

If YYY is finite, we have

E(X)=∑yi∈YE(X∣Y=yi)p(Y=yi)E(X) = \sum_{y_{i} \in \mathcal{Y}} E(X \mid Y = y_{i}) p(Y = y_{i}) E(X)=yi​∈Y∑​E(X∣Y=yi​)p(Y=yi​)

where Y\mathcal{Y}Y is the alphabet of YYY.

Thus we obtain:

E[Rt+1∣St=s]=∑a∈A(s)π(a∣s)E[Rt+1∣St=s,At=a]E[R_{t+1} \mid S_{t} = s] = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) E[R_{t+1} \mid S_{t} = s, A_{t} = a] E[Rt+1​∣St​=s]=a∈A(s)∑​π(a∣s)E[Rt+1​∣St​=s,At​=a]

where AtA_{t}At​ is finite.

  1. From the definition of expectation, E[Rt+1∣St=s,At=a]=p(r∣s,a)rE[R_{t+1} \mid S_{t} = s, A_{t} = a] = p(r \mid s, a) rE[Rt+1​∣St​=s,At​=a]=p(r∣s,a)r, leading to

    ∑a∈A(s)π(a∣s)E[Rt+1∣St=s,At=a]=∑a∈A(s)π(a∣s)∑rp(r∣s,a)r\sum_{a \in \mathcal{A}(s)} \pi(a \mid s) E[R_{t+1} \mid S_{t} = s, A_{t} = a] = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \sum_{r} p(r \mid s, a) r a∈A(s)∑​π(a∣s)E[Rt+1​∣St​=s,At​=a]=a∈A(s)∑​π(a∣s)r∑​p(r∣s,a)r

A more verbose version of deduction is:

First, consider the definition of expectation:

E[Rt+1∣St=s]≜∑rp(r∣s)r\mathbb{E} \left[ R_{t+1} \mid S_{t} = s \right] \triangleq \sum_{r} p(r \mid s) r E[Rt+1​∣St​=s]≜r∑​p(r∣s)r

Look at the formula for marginal probability,

p(r∣s)=∑a∈A(s)p(r,a∣s)p(r \mid s) = \sum_{a \in A(s)} p(r, a \mid s) p(r∣s)=a∈A(s)∑​p(r,a∣s)

p(r,a∣s)=p(r∣a,s)⋅p(a∣s)p(r, a \mid s) = p(r \mid a, s) \cdot p(a \mid s) p(r,a∣s)=p(r∣a,s)⋅p(a∣s)

And in RL context, p(a∣s)p(a \mid s)p(a∣s) is often written as π(a∣s)\pi(a \mid s)π(a∣s).

Therefore,

p(r∣s)=∑a∈A(s)π(a∣s)⋅p(r∣a,s)p(r \mid s) = \sum_{a \in A(s)} \pi(a \mid s) \cdot p(r \mid a, s) p(r∣s)=a∈A(s)∑​π(a∣s)⋅p(r∣a,s)

Replace p(r∣s)p(r \mid s)p(r∣s) in the definition of expectation with ∑a∈A(s)π(a∣s)⋅p(r∣a,s)\sum_{a \in A(s)} \pi(a \mid s) \cdot p(r \mid a, s)∑a∈A(s)​π(a∣s)⋅p(r∣a,s), we get

Second term: the (discounted) mean of future rewards

First, calculate the mean of future rewards

E[Gt+1∣St=s]=∑s′E[Gt+1∣St=s,St+1=s′]p(s′∣s)=∑s′E[Gt+1∣St+1=s′]p(s′∣s)=∑s′vπ(s′)p(s′∣s)=∑s′vπ(s′)∑ap(s′∣s,a)π(a∣s)\begin{aligned} \mathbb{E} \left[ G_{t+1} \mid S_{t} = s \right] &= \sum_{s'} \mathbb{E} \left[ G_{t+1} \mid S_{t} = s, S_{t+1} = s' \right] p \left( s' \mid s \right) \\ &= \sum_{s'} \mathbb{E} \left[ G_{t+1} \mid S_{t+1} = s' \right] p \left( s' \mid s \right) \\ &= \sum_{s'} v_{\pi} \left( s' \right) p \left( s' \mid s \right) \\ &= \sum_{s'} v_{\pi} \left( s' \right) \sum_{a} p \left( s' \mid s, a \right) \pi (a \mid s) \end{aligned} E[Gt+1​∣St​=s]​=s′∑​E[Gt+1​∣St​=s,St+1​=s′]p(s′∣s)=s′∑​E[Gt+1​∣St+1​=s′]p(s′∣s)=s′∑​vπ​(s′)p(s′∣s)=s′∑​vπ​(s′)a∑​p(s′∣s,a)π(a∣s)​

Then we multiply it with a dicount factor γ\gammaγ. For simplicity, we say the second term γE[Gt+1∣St=s]\gamma\mathbb{E}[G_{t+1}|S_{t}=s]γE[Gt+1​∣St​=s] is “the mean of future rewards” also it’s discounted.

Explanation

The transotion of the first line comes from the law of total expectation as well.

The transotion of the second line comes from the Markov property. Recall that reward rtr_{t}rt​ is defined to only rely on sts_{t}st​ and ata_{t}at​:

p(rt+1∣st,at,st−1,at−1,…,s0,a0)=p(rt+1∣st,at)p(r_{t+1} \mid s_{t}, a_{t}, s_{t-1}, a_{t-1}, \ldots, s_{0}, a_{0}) = p(r_{t+1} \mid s_{t}, a_{t}) p(rt+1​∣st​,at​,st−1​,at−1​,…,s0​,a0​)=p(rt+1​∣st​,at​)

we obtaini that

E[Rt+1∣st,at,st−1,at−1,…,s0,a0]≜∑rt+1∈Rt+1p(rt+1∣st,at,st−1,at−1,…,s0,a0)⋅rt+1=∑rt+1∈Rt+1p(rt+1∣st,at)⋅rt+1≜E[Rt+1∣st,at].\begin{aligned} \mathbb{E}[R_{t+1} \mid s_{t}, a_{t}, s_{t-1}, a_{t-1}, \ldots, s_{0}, a_{0}] &\triangleq \sum_{r_{t+1} \in R_{t+1}} p(r_{t+1} \mid s_{t}, a_{t}, s_{t-1}, a_{t-1}, \ldots, s_{0}, a_{0}) \cdot r_{t+1} \\ &= \sum_{r_{t+1} \in R_{t+1}} p(r_{t+1} \mid s_{t}, a_{t}) \cdot r_{t+1} \\ &\triangleq \mathbb{E}[R_{t+1} \mid s_{t}, a_{t}]. \end{aligned} E[Rt+1​∣st​,at​,st−1​,at−1​,…,s0​,a0​]​≜rt+1​∈Rt+1​∑​p(rt+1​∣st​,at​,st−1​,at−1​,…,s0​,a0​)⋅rt+1​=rt+1​∈Rt+1​∑​p(rt+1​∣st​,at​)⋅rt+1​≜E[Rt+1​∣st​,at​].​

since

Gt=Rt+1+Rt+2+2Rt+3+…G_{t} = R_{t+1}+R_{t+2}+^2 R_{t+3}+\dots Gt​=Rt+1​+Rt+2​+2Rt+3​+…

we have

E[Gt+1∣St=s,St+1=s′]=E[Rt+1+γRt+2+γ2Rt+3+…∣St=s,St+1=s′]=E[Rt+1+γRt+2+γ2Rt+3+…∣St+1=s′]\begin{aligned} \mathbb{E}[G_{t+1} \mid S_{t} = s, S_{t+1} = s'] &= \mathbb{E}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \ldots \mid S_{t} = s, S_{t+1} = s'] \\ &= \mathbb{E}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \ldots \mid S_{t+1} = s'] \end{aligned} E[Gt+1​∣St​=s,St+1​=s′]​=E[Rt+1​+γRt+2​+γ2Rt+3​+…∣St​=s,St+1​=s′]=E[Rt+1​+γRt+2​+γ2Rt+3​+…∣St+1​=s′]​

Bellman equation

Basic derivation，we have the Bellman equation

vπ(s)=E[Rt+1∣St=s]+γE[Gt+1∣St=s],=∑aπ(a∣s)∑rp(r∣s,a)r⏟mean of immediate rewards+γ∑s′π(a∣s)∑ap(s′∣s,a)vπ(s′),⏟(discounted) mean of future rewards=∑aπ(a∣s)[∑rp(r∣s,a)r+γ∑s′p(s′∣s,a)vπ(s′)],∀s∈S.\begin{aligned}v_\pi(s)&=\mathbb{E}\left[R_{t+1}\mid S_t=s\right]+\gamma\mathbb{E}\left[G_{t+1}\mid S_t=s\right],\\&=\underbrace{\textcolor{skyblue}{\sum_a\pi(a\mid s)\sum_rp(r\mid s,a)r}}_{\text{mean of immediate rewards}}+\underbrace{\textcolor{hotpink}{\gamma\sum_{s^{\prime}}\pi(a\mid s)\sum_ap\left(s^{\prime}\mid s,a\right)v_\pi\left(s^{\prime}\right)},}_{\text{(discounted) mean of future rewards}}\\&=\sum_a\pi(a\mid s)\left[\sum_rp(r\mid s,a)r+\gamma\sum_{s^{\prime}}p\left(s^{\prime}\mid s,a\right)v_\pi\left(s^{\prime}\right)\right],\quad\forall s\in\mathcal{S}.\end{aligned} vπ​(s)​=E[Rt+1​∣St​=s]+γE[Gt+1​∣St​=s],=mean of immediate rewardsa∑​π(a∣s)r∑​p(r∣s,a)r​​+(discounted) mean of future rewardsγs′∑​π(a∣s)a∑​p(s′∣s,a)vπ​(s′),​​=a∑​π(a∣s)[r∑​p(r∣s,a)r+γs′∑​p(s′∣s,a)vπ​(s′)],∀s∈S.​

It is a fundamental tool for designing and analyzing reinforcement learning algorithms.

Remarks：

  • vπ(s)v_\pi(s)vπ​(s) and vπ(s′)v_\pi\left(s^{\prime}\right)vπ​(s′) are state values to be calculated. Bootstrapping！
  • π(a∣s)\pi(a\mid s)π(a∣s) is a given policy. Solving the equation is called policy evaluation.
  • p(r∣s,a)p(r\mid s,a)p(r∣s,a) and p(s′∣s,a)p\left(s^{\prime}\mid s,a\right)p(s′∣s,a) represent the dynamic model（environment model）.

p(s′=s3∣s1,a3)=1andp(s′≠s3∣s1,a3)=0.p(s' = s_3 | s_1, a_3) = 1 \quad \text{and} \quad p(s' \neq s_3 | s_1, a_3) = 0. p(s′=s3​∣s1​,a3​)=1andp(s′=s3​∣s1​,a3​)=0.

p(r=0∣s1,a3)=1andp(r≠0∣s1,a3)=0.p(r = 0 | s_1, a_3) = 1 \quad \text{and} \quad p(r \neq 0 | s_1, a_3) = 0. p(r=0∣s1​,a3​)=1andp(r=0∣s1​,a3​)=0.
