Stochastic approximation（SA）

Stochastic approximation (SA):

  • SA refers to a broad class of stochastic iterative algorithms solving root-finding or optimization problems.
  • Compared to many other root-finding algorithms such as gradient-based methods, SA is powerful in the sense that it does not require knowing the expression of the objective function nor its derivative.

Robbins-Monro (RM) algorithm:

  • This is a pioneering work in the field of stochastic approximation.
  • The famous stochastic gradient descent algorithm is a special form of the RM algorithm.
  • It can be used to analyze the mean estimation algorithms introduced in the post.

Problem statement

Suppose we would like to find the root of the equation

g(w)=0,g(w) = 0, g(w)=0,

where w∈Rw \in \mathbb{R}w∈R is the variable to be solved and g:R→Rg : \mathbb{R} \to \mathbb{R}g:R→R is a function.

  • Many problems can be eventually converted to this root finding problem. For example, suppose J(w)J(w)J(w) is an objective function to be minimized. Then, the optimization problem can be converted to

g(w)=∇wJ(w)=0g(w) = \nabla_w J(w) = 0 g(w)=∇w​J(w)=0

  • Note that an equation like g(w)=cg(w) = cg(w)=c with ccc as a constant can also be converted to the above equation by rewriting g(w)−cg(w) - cg(w)−c as a new function.

How to calculate the root of g(w)=0g(w) = 0g(w)=0?

  • Model-based: If the expression of ggg is known, there are many numerical algorithms that can solve this problem.
  • Model-free: What if the expression of the function ggg is unknown? For example, the function is represented by an artificial neural network.

The Robbins-Monro Algorithm

The Robbins-Monro (RM) algorithm that can solve this problem is as follows:

wk+1=wk−akg~(wk,ηk),k=1,2,3,…w_{k+1} = w_k - a_k \tilde{g}(w_k, \eta_k), \quad k = 1, 2, 3, \ldots wk+1​=wk​−ak​g~​(wk​,ηk​),k=1,2,3,…

where

  • wkw_kwk​ is the kkkth estimate of the root
  • g~(wk,ηk)=g(wk)+ηk\tilde{g}(w_k, \eta_k) = g(w_k) + \eta_kg~​(wk​,ηk​)=g(wk​)+ηk​ is the kkkth noisy observation
    • Why noise here? For example, consider a random sampling xxx of XXX.
  • aka_kak​ is a positive coefficient.

This algorithm relies on data instead of model:

  • Input sequence: {wk}\{w_k\}{wk​}
  • Output sequence (noisy): {g~(wk,ηk)}\{\tilde{g}(w_k, \eta_k)\}{g~​(wk​,ηk​)}

Philosophy: without model, we need data!

  • The function g(w)g(w)g(w) is viewed as a black box.
  • The model here refers to the expression of the function.

Theorem (Robbins-Monro Theorem)

In the Robbins-Monro algorithm, if

  1. 0<c1≤∇wg(w)≤c20 < c_1 \leq \nabla_w g(w) \leq c_20<c1​≤∇w​g(w)≤c2​ for all www;
  2. ∑k=1∞ak=∞\sum_{k=1}^\infty a_k = \infty∑k=1∞​ak​=∞ and ∑k=1∞ak2<∞\sum_{k=1}^\infty a_k^2 < \infty∑k=1∞​ak2​<∞;
  3. E[ηk∣Hk]=0\mathbb{E}[\eta_k|\mathcal{H}_k] = 0E[ηk​∣Hk​]=0 and E[ηk2∣Hk]<∞\mathbb{E}[\eta_k^2|\mathcal{H}_k] < \inftyE[ηk2​∣Hk​]<∞;

where Hk={wk,wk−1,…}\mathcal{H}_k = \{w_k, w_{k-1}, \ldots\}Hk​={wk​,wk−1​,…}, then wkw_kwk​ converges with probability 1 (w.p.1w.p.1w.p.1) to the root w∗w^*w∗ satisfying g(w∗)=0g(w^*) = 0g(w∗)=0.

Explanation of the Three Conditions

  1. Condition 1: 0<c1≤∇wg(w)≤c20 < c_1 \leq \nabla_w g(w) \leq c_20<c1​≤∇w​g(w)≤c2​ for all www

    • ggg should be monotonically increasing, which ensures that the root of g(w)=0g(w) = 0g(w)=0 exists and is unique.
    • The gradient is bounded from above.
    • This condition is not strict. Consider the example g(w)=∇wJ(w)=0g(w) = \nabla_w J(w) = 0g(w)=∇w​J(w)=0. This condition requires that g(w)g(w)g(w) is convex.

  2. Condition 2: ∑k=1∞ak=∞\sum_{k=1}^\infty a_k = \infty∑k=1∞​ak​=∞ and ∑k=1∞ak2<∞\sum_{k=1}^\infty a_k^2 < \infty∑k=1∞​ak2​<∞

    • ∑k=1∞ak2<∞\sum_{k=1}^\infty a_k^2 < \infty∑k=1∞​ak2​<∞ ensures that aka_kak​ converges to zero as k→∞k \to \inftyk→∞.
    • ∑k=1∞ak=∞\sum_{k=1}^\infty a_k = \infty∑k=1∞​ak​=∞ ensures that aka_kak​ does not converge to zero too fast.

  3. Condition 3: E[ηk∣Hk]=0\mathbb{E}[\eta_k | \mathcal{H}_k] = 0E[ηk​∣Hk​]=0 and E[ηk2∣Hk]<∞\mathbb{E}[\eta_k^2 | \mathcal{H}_k] < \inftyE[ηk2​∣Hk​]<∞

    • A special yet common case is that {ηk}\{\eta_k\}{ηk​} is an iid stochastic sequence satisfying E[ηk]=0\mathbb{E}[\eta_k] = 0E[ηk​]=0 and E[ηk2]<∞\mathbb{E}[\eta_k^2] < \inftyE[ηk2​]<∞. The observation error ηk\eta_kηk​ is not required to be Gaussian.

Since

  • If ak→0a_k \to 0ak​→0, then akg~(wk,ηk)→0a_k \tilde{g}(w_k, \eta_k) \to 0ak​g~​(wk​,ηk​)→0 and hence wk+1−wk→0w_{k+1} - w_k \to 0wk+1​−wk​→0.
  • We need the fact that wk+1−wk→0w_{k+1} - w_k \to 0wk+1​−wk​→0 if wkw_kwk​ converges eventually.
  • If wk→w∗w_k \to w^*wk​→w∗, g(wk)→0g(w_k) \to 0g(wk​)→0 and g~(wk,ηk)\tilde{g}(w_k, \eta_k)g~​(wk​,ηk​) is dominated by ηk\eta_kηk​.
  • Second, ∑k=1∞ak=∞\sum_{k=1}^{\infty} a_k = \infty∑k=1∞​ak​=∞ indicates that aka_kak​ should not converge to zero too fast.

Why is this condition important?

Summarizing w2=w1−a1g~(w1,η1)w_2 = w_1 - a_1 \tilde{g}(w_1, \eta_1)w2​=w1​−a1​g~​(w1​,η1​), w3=w2−a2g~(w2,η2),⋯ ,wk+1=wk−akg~(wk,ηk)w_3 = w_2 - a_2 \tilde{g}(w_2, \eta_2),\cdots, w_{k+1} = w_k - a_k \tilde{g}(w_k, \eta_k)w3​=w2​−a2​g~​(w2​,η2​),⋯,wk+1​=wk​−ak​g~​(wk​,ηk​) leads to

Suppose w∞=w∗w_\infty = w^*w∞​=w∗. If ∑k=1∞ak<∞\sum_{k=1}^{\infty} a_k < \infty∑k=1∞​ak​<∞, then ∑k=1∞akg~(wk,ηk)\sum_{k=1}^{\infty} a_k \tilde{g}(w_k, \eta_k)∑k=1∞​ak​g~​(wk​,ηk​) may be bounded. Apply RM to Mean Estimation

Recall that

wk+1=wk+αk(xk−wk).w_{k+1} = w_k + \alpha_k (x_k - w_k). wk+1​=wk​+αk​(xk​−wk​).

is the mean estimation algorithm introduced at the beginning of this lecture.

  • If αk=1/k\alpha_k = 1/kαk​=1/k, then wk+1=1/k∑i=1kxiw_{k+1} = 1/k \sum_{i=1}^k x_iwk+1​=1/k∑i=1k​xi​.
  • If αk\alpha_kαk​ is not 1/k1/k1/k, the convergence was not analyzed.

Next, we show that this algorithm is a special case of the RM algorithm. Then, its convergence naturally follows.

  1. Consider a function:

g(w)≐w−E[X].g(w) \doteq w - \mathbb{E}[X]. g(w)≐w−E[X].

Our aim is to solve g(w)=0g(w) = 0g(w)=0. If we can do that, then we can obtain E[X]\mathbb{E}[X]E[X].

  • Mean estimation (i.e., finding E[X]\mathbb{E}[X]E[X]) is formulated as a root-finding problem (i.e., solving g(w)=0g(w) = 0g(w)=0).
  • Question: Do we know the expression of g(w)g(w)g(w) here?
  1. The observation we can get is

g~(w,x)≐w−x,\tilde{g}(w,x) \doteq w - x, g~​(w,x)≐w−x,

because we can only obtain samples of XXX. Note that

g~(w,η)=w−x=w−x+E[X]−E[X]=(w−E[X])+(E[X]−x)≐g(w)+η,\tilde{g}(w,\eta) = w - x = w - x + \mathbb{E}[X] - \mathbb{E}[X] = (w - \mathbb{E}[X]) + (\mathbb{E}[X] - x) \doteq g(w) + \eta, g~​(w,η)=w−x=w−x+E[X]−E[X]=(w−E[X])+(E[X]−x)≐g(w)+η,

  1. The RM algorithm for solving g(x)=0g(x) = 0g(x)=0 is

wk+1=wk−αkg~(wk,ηk)=wk−αk(wk−xk),w_{k+1} = w_k - \alpha_k \tilde{g}(w_k, \eta_k) = w_k - \alpha_k (w_k - x_k), wk+1​=wk​−αk​g~​(wk​,ηk​)=wk​−αk​(wk​−xk​),

which is exactly the mean estimation algorithm.

The convergence naturally follows.

Theorem (Dvoretzky’s Theorem)

wk+1=(1−αk)wk+βkηk,w_{k+1} = (1 - \alpha_k) w_k + \beta_k \eta_k, wk+1​=(1−αk​)wk​+βk​ηk​,

  1. ∑k=1∞αk=∞,∑k=1∞αk2<∞;∑k=1∞βk2<∞\sum_{k=1}^\infty \alpha_k = \infty, \sum_{k=1}^\infty \alpha_k^2 < \infty; \sum_{k=1}^\infty \beta_k^2 < \infty∑k=1∞​αk​=∞,∑k=1∞​αk2​<∞;∑k=1∞​βk2​<∞, uniformly w.p.1;
