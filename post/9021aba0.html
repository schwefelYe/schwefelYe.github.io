Bellman Optimality Equation, which is a special Bellman equation whose corresponding policy is optimal.

Optimal policy

The state value could be used to evaluate if a policy is good or not: if

vπ1(s)≥vπ2(s)v_{\pi_1}(s) \geq v_{\pi_2}(s) vπ1​​(s)≥vπ2​​(s)

for all s∈Ss \in Ss∈S,

then π1\pi_1π1​ is “better” than π2\pi_2π2​.

Definition

A policy π∗\pi^*π∗ is optimal if

vπ∗(s)≥vπ(s)v_{\pi^*}(s) \geq v_{\pi}(s) vπ∗​(s)≥vπ​(s)

for all sss and for any other policy π\piπ.

The definition leads to many questions:

  • Does the optimal policy exist?
  • Is the optimal policy unique?
  • Is the optimal policy stochastic or deterministic?
  • How to obtain the optimal policy?

To answer these questions, introducing Bellman optimality equation.

Bellman Optimality Equation（BOE）

The BOE is the tool to analyze optimal state values and optimal policies. Two forms of BOE are as follows:

Elementwise form

  • p(r∣s,a),p(s′∣s,a)p(r|s,a), p(s'|s,a)p(r∣s,a),p(s′∣s,a) are known.

When the state values don’t change too much, i.e., they have converged, the state value function is the optimal state value function, and the policy it adopts during that computation is the optimal policy.

Maximize on the right-hand side of BOE

Here we talk about how to solve BOE. In practice, we need to deal with the matrix-vector form since that is what we’re faced with. But since each row in the matrix is actually a vector of the elementwise form, we start with the element form.

Example 1

In fact, we can turn the problem into “solve the optimal π\piπ on the right-hand side”. Let’s look at one example first:

Example 1. Consider two unknown variables x,y∈Rx, y \in \mathbb{R}x,y∈R that satisfy

x=max⁡y∈R(2x−1−y2).x = \max_{y \in \mathbb{R}} \left( 2x - 1 - y^2 \right). x=y∈Rmax​(2x−1−y2).

Regardless of the value of xxx, we always have max⁡y(2x−1−y2)=2x−1\max_{y} \left( 2x - 1 - y^2 \right) = 2x - 1maxy​(2x−1−y2)=2x−1, where the maximum is achieved when y=0y = 0y=0. The second step is to solve xxx. When y=0y = 0y=0, the equation becomes x=2x−1x = 2x - 1x=2x−1, which leads to x=1x = 1x=1. Therefore, y=0y = 0y=0 and x=1x = 1x=1 are the solutions of the equation.

Now that we know the solution of BOE is to maximize the right-hand side, and we know how to do it as well — just select the action with the largest action value. But we don’t know action value or state value at this time, so this method doesn’t work.

In fact, the solution of BOE derives from the contraction mapping theorem (see later) on the matrix-vector form. That’s an iterative method.

So why do we introduce (1) here? The reason is that during every iteration, for every state sss, the action value will already have been known, so we can use (1) to get the maximized right-hand side, which is the maximized v(s)v(s)v(s)!
